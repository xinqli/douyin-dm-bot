## 🔮 核心功能

### 抖音账号托管

- 支持多个抖音账号统一接入与管理
- 账号在线状态、回复数据一目了然
- 支持随时启用 / 停用单个账号
- 服务到期时间清晰可见，续费方便

### 私信自动回复

- **进入会话自动应答**：用户发来私信后，按预设话术第一时间回复
- **多条文案配置**：欢迎语、引流话术、常见问题引导等，灵活搭配
- **智能防刷**：同一用户触发回复后自动静音，避免重复打扰
- **7×24 无人值守**：夜间、节假日也能及时承接私信流量

适用场景：活动引流、私信入口统一话术、夜间自动接待。

### 后台私信客服

- **网页端工作台**：无需反复登录抖音 App，在后台集中处理私信
- **会话列表管理**：按账号筛选，支持搜索昵称、抖音号
- **完整聊天记录**：往来消息清晰展示，方便跟进高意向用户
- **一键代发回复**：客服在后台输入内容，系统自动代为发送
- **多人协作**：适合团队分工，共管多个抖音号

适用场景：自动回复后转人工、白天重点跟进、多客服协同。

### 服务到期管理

- 按**抖音账号**授权使用时长
- **不限制回复条数**（在平台规则允许范围内）
- 到期后自动停止自动回复与客服代发
- 支持续费延期，多账号可单独或打包开通

## 📋 使用流程

1. **准备抖音账号**：完成实名认证（建议蓝 V 或稳定老号）
2. **开通服务**：获取后台账号与开通方案
3. **授权托管**：按后台指引完成抖音账号授权
4. **配置话术**：设置自动回复文案，按需开启 / 关闭
5. **验证效果**：用小号发私信，确认对方能正常收到回复
6. **人工跟进**：在「私信客服」中查看会话，随时人工介入

## ❓ 常见问题

**Q：显示发送成功，对方没收到？**  
A：优先确认账号是否**已完成实名**；若已实名仍异常，请检查账号状态或提交 Issue 反馈。

**Q：自动回复开了但不生效？**  
A：检查账号是否启用、服务是否在有效期内。

**Q：后台客服发不出去？**  
A：请确认账号处于启用且服务未到期状态。

**Q：未实名账号能用吗？**  
A：**不建议。** 未实名或风控账号，平台可能**吞掉**出站私信，业务号请务必实名。

**Q：能否对接 AI / 知识库智能对话？**  
A：当前版本支持**固定话术自动回复** + **人工客服**；智能对话可按需求定制。
